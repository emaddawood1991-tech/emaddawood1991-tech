# 06 - Teach This Workflow

Use this file if you want to teach the automation to other people.

## Simple Video Structure

### Hook

```text
I built an AI automation that reads Meta Ads performance, finds losing ads, and creates a daily media buyer report.
```

### Problem

```text
Media buyers spend too much time checking Ads Manager manually every day.
```

### Solution

```text
We connect Meta Ads, n8n, and AI. The system reads the numbers, applies rules, and creates a recommendation report.
```

### Safety

```text
The workflow starts in report-only mode, so it does not change live ads until a human approves.
```
