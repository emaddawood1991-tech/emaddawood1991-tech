# 04 - Automation Ops

This is the daily operating process.

## Daily Review

Every morning:

1. Run yesterday's report.
2. Review total spend.
3. Check leads, purchases, CPC, CTR, CPM, and ROAS.
4. Read AI summary.
5. Approve or reject recommended actions.
6. Write down what changed.

## Weekly Review

Every week:

1. Compare the last 7 days with the previous 7 days.
2. Find winning creative angles.
3. Create new ad variations.
4. Kill weak angles.
5. Review landing page performance.
6. Check tracking quality.

## Decision Rules

Pause ads when:

- Spend is above your threshold.
- There are zero conversions.
- CPC is too high.
- CTR is weak.
- Frequency is too high.

Scale ads when:

- ROAS is strong.
- CPA is below target.
- Conversion volume is stable.
- Creative quality is still fresh.

## Human Judgment Still Matters

Do not pause an ad only because one metric looks bad.

Check:

- Testing period
- Budget size
- Audience size
- Campaign objective
- Tracking quality
